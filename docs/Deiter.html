Deiter

type Deiter<T> = { next : () -> ?T; next_back : () -> ?T }

Double Ended Iterator Type

public func rev<T>(deiter : Deiter<T>) : Deiter<T>

Returns an iterator that iterates over the elements in reverse order.

Examples

public func fromArray<T>(array : [T]) : Deiter<T>

public func toArray<T>(deiter : Deiter<T>) : [T]

public func fromArrayMut<T>(array : [var T]) : Deiter<T>

public func toArrayMut<T>(deiter : Deiter<T>) : [var T]

public func toIter<T>(iter : Iter.Iter<T>) : Iter.Iter<T>

Deiter are interchangable with Iter types so there is no need to convert them. This function casts a Deiter to an Iter type. So you won't be able to use the next_back function after calling this function. So if you want to get the values in reverse order you need to use the rev function before calling this function.
